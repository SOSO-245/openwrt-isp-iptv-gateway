# 安全与脱敏说明

## 发布脱敏原则

公开仓库不应包含：

- 私人密码、SSH 端口、账号。
- 家庭内网固定 IP、NAS 私有地址。
- 运行时节目单备份、私人台标库完整备份。
- 路由器 `/etc/shadow`、`/etc/passwd`、Dropbear key。
- 个人运行日志和抓包文件。

本仓库发布前使用敏感词扫描检查过公开目录。

## 默认账号

插件默认管理账号为：

```text
admin / admin
```

首次登录后请立即修改。

## 网络风险

本插件会修改 OpenWrt 网络、路由、`udpxy` 和抓包相关配置。请在确认可恢复路由器的前提下使用。

## 抓包风险

抓包功能只应在你有权管理的家庭网络和自有设备上使用。不要采集、保存或传播无授权的第三方通信内容。

## 报告问题

公开 issue 请不要贴：

- 密码、token、cookie。
- 真实公网 IP。
- 完整抓包文件。
- 含个人频道订阅或家庭拓扑的敏感截图。

建议先做脱敏后再提交。
